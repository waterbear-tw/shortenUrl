## 使用方式

1. 安裝 Node.js 以及 npm
2. clone 此專案

```
git clone "https://github.com/waterbear-tw/restaurant_express.git"
```

3. 進入專案資料夾安裝 package.json 中的套件

```
npm install
```

4. 啟動專案

```
npm run start
```

5. 開啟瀏覽器輸入 localhost:3000 即可執行網站
6. 在終端機按下 ctrl + c 結束執行。
